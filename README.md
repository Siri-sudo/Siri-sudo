## Projects

**[eBPF Kubernetes IDS](https://github.com/Siri-sudo/ebpf-k8s-ids)**
Kernel-space IDS that hooks into K8s worker-node tracepoints to detect port scans in real time; no userspace agent. Benchmarked against Snort: 3.6% faster, 5% less RAM. MSc dissertation.
`eBPF` `Linux` `Kubernetes`

**[Detectory](https://github.com/Siri-sudo/detectory)**
Automated web vulnerability scanner. Chains Nmap and custom Python scripts to surface OWASP Top 10 issues; exports CVSS-scored PDF reports.
`Python` `Nmap` `Burp Suite`
